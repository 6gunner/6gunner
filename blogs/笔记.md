一、翻牌结构：

两个div, 但是都利用伪元素来进行处理,

div::before

div::after

这样相当于4个div；

```
div
	div0
	div1

```

利用zIndex，将0放在前面；

2、before的top：0， bottom: 50%
after的top: 50%, bottom: 0

3、利用transform将back的after翻转上去：

```
&.back {

&::before {

z-index: 1;

}

&::after {

z-index: 3;

transform: perspective(160px) rotateX(180deg);

transform-origin: 50% 0;

transform-style: preserve-3d;



}

}
```

css3一些属性：
perspective 是 CSS 3D 变换中一个很重要的属性，它定义了观察者与 z=0 平面的距离，用来产生 3D 效果的视觉深度。

让我用简单的方式解释：

- perspective 值越小，3D 效果越明显（观察者离物体越近）

- perspective 值越大，3D 效果越微弱（观察者离物体越远）

二、组件设计：

如果是时钟的话 ：怎么和当前时间同步？

当前只设计了一个翻转的样式，能够实现动画效果，如果想让它能够和时钟一样，就得提取组件，然后支持传参。
时钟的话，

- 组件需要支持设置最大值（59，23），最小值（0）
- 然后需要一个设计定时器，来触发我们的动画；
- 需要能够设置初始值，
- 最好能够定时矫正时间，可能因为interval有误差；

如果是倒计时的话，怎么计算时差然后展示出来？
同上，倒计时的组件需要支持：

- 组件需要支持设置最大值（59，23），最小值（0）
- 顺序不同，需要从59递减到0
- 需要能够设置初始值
